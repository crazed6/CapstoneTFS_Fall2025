# Kynetic Flux

### Capstone 2025 — Toronto Film School

Kynetic Flux is a game project developed as a capstone submission by graduates of the Toronto Film School's 2025 program. This repository represents the collaborative work completed during the program's final term.


### Project Status

This project is currently **unmaintained**. Development concluded with the graduation of the original team, and there are no active contributors or maintainers at this time. The repository is preserved as-is for archival, portfolio, and reference purposes.
